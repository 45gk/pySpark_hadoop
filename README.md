1. Как пользоваться:

* Jupyter Lab (PySpark 3): Перейдите в браузере на localhost:8888. В консоли Docker вы увидите ссылку с токеном для входа.

* HDFS Web UI: Доступен на localhost:9870. Вы сможете видеть свои файлы через браузер.

* Связка: В ноутбуке PySpark будет готов к работе. Чтобы сохранить данные в HDFS, используйте путь: df.write.save("hdfs://namenode:9000/my_data").
